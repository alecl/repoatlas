# Demo Directory

This directory contains vendored Java demo projects used as analyzer test fixtures. These are **not** part of the RepoAtlas codebase.

- Do not modify these files when working on RepoAtlas features or bug fixes.
- Do not treat Java code here as project source — it exists only to exercise the analyzer pipeline.
- Do not include these files in searches for RepoAtlas implementation code.
- Do not run linters, formatters, or type checkers against this directory.
